What's locked here

  • IA: /research gains a tab navigation (Watchlist · Insights); /research/insights is the canonical URL for the new tab. Sidebar item stays "Research" (single slot).
  • Modal-route pattern: each observation opens via ?insight=<id> drawer (per components.md §0.4b).
  • Search bar (above filters): full-text search across narrative, symbol, pattern_type, linked trade_id. Keyboard shortcut ⌘K / Ctrl+K focuses it. Search query persists as ?q= URL param so deep-links survive reload. Filters and search compose (AND).
  • Filter chips (below search): pattern_type, symbol, date_range, has_trade_link, recall_state. Active chips persist in URL.
  • Stats row: 4 cards — total observations, distinct patterns, recall-off (trustworthy %), linked-to-trades %.
  • Cross-symbol aggregation: top patterns grouped by pattern_type with count + symbol distribution + recall-state breakdown. Above the list, below the stats.
  • Recall-state visibility: every observation row shows the recall pill (off=success-tinted "trustworthy", on=warn-tinted "potentially biased"). This is the single most important meta-signal — without it, observations are mixed signal/noise.
  • Linked artefacts in drawer: trade(s) where the pattern was matched, brief version that recorded it, weekly PDF that contains it.
  • Export CSV: per-filter-selection, FR50 contract.
