My read

Lucide — 1.5px stroke matches the operational/Bloomberg lineage; native octagon-x for kill-switch and gauge for Risk Cockpit; tree-shakes per import; ISC license.

Heroicons outline — same stroke weight, looks tighter (more rounded corners), but missing octagon-x (we'd substitute no-symbol which is a circular "Ø" — wrong semantic for kill-switch) and gauge (closest is chart-bar — generic).

Heroicons solid — fills feel heavy on dark; clashes with the "fine instrument" reading the catalogue is going for. Risk: P&L delta arrow looks like a chunky badge instead of a directional cue.

→ My recommendation: Lucide cherry-picked
