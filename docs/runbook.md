# iguanatrader — Runbook

> Populated when the project adopts the BMAD + OpenSpec workflow. See
> `.ai-playbook/specs/dispatcher-chain.md` for the universal flow, and
> extend here with project-specific deviations.

## Flow

1. **Discovery (BMAD)** — PRD / personas / outcome / KPIs / FRs.
2. **Human slicing** — break PRD into OpenSpec-shaped changes.
3. **Per change (OpenSpec)**:
   - `/opsx:propose` → `specs || design` → `tasks` → `openspec apply` → `openspec archive`.
4. **Retros** — post-archive + weekly + monthly (`.ai-playbook/specs/retrospective-cadence.md`).
