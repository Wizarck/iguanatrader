- [x] 2.6 Create `apps/openbb-sidecar/src/openbb_sidecar/routes/health.py` exporting `router: APIRouter`. `GET /health` returns 200 with body `{"status": "ok", "openbb_loadable": OpenBBFacade().is_ready(), "version": __version__}`.
